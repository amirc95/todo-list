# ToDo List - Laravel API

## Description

This Laravel API offers basic CRUD functions for a ToDo - List.

## Getting started

Setting up a Laravel development environment can be overwhelming at first. Therefore, some tools are being used to make this process more straightforward.

### Requirements

- [Docker](https://www.docker.com/) (& docker-compose) is installed
- [Ddev](https://ddev.readthedocs.io/en/latest/) is installed

### Start up

**NOTE: There is already an excisting build for this projects React app inside the public folder. Meaning you can use the project (API+UI) by just launching the Laravel application with the following steps:**

- Open a terminal in the api directory
- Run `bash setup.sh` to configure the app
- Run `ddev launch` to start the app
- Open the app in your browser: 
<br/>http://todo-api.ddev.site/app
